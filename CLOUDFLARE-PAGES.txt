Cloudflare Pages + Wrangler CLI
================================

This site is static HTML/JS. Deploy from your machine with Wrangler (recommended for you).

1) Prerequisites
   - Node.js 20.3 or newer (Wrangler requires it)
   - A Cloudflare account

2) One-time setup
   npm install
   npx wrangler login
   (Browser opens — authorize Wrangler for your account.)

3) Deploy (every time you want to publish)
   npm run deploy

   That runs: wrangler pages deploy
   It reads ./wrangler.toml — static files are the repo root (pages_build_output_dir = ".").

4) Project name
   wrangler.toml has: name = "paystub-calculator-web"
   Your Cloudflare Pages project name should match, OR run once with an explicit name:
   npx wrangler pages deploy --project-name=YOUR_PROJECT_NAME

5) Do NOT run
   npx wrangler deploy
   That is for Workers only, not Pages. Use "wrangler pages deploy" (or npm run deploy).

---

Optional: Git-connected builds in Cloudflare
--------------------------------------------
If the dashboard still runs a custom "Deploy command", clear it or set it to:
  npx wrangler pages deploy
Never use: npx wrangler deploy
